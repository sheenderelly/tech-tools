# DevTools

A small collection of browser-based developer tools.

## Tools

- **Text**: Change Case (UPPERCASE / lowercase / Sentence case), Kana Width converter (full-width ↔ half-width).
- **Number**: Phone number formatter for JP (+81) and PH (+63), with country-code / hyphen / space toggles.
- **Salesforce**: Create Icons — paste an SVG and download three PNGs (16×16 and 32×32 legacy custom-tab icons + 480×480 SLDS-spec icon with a configurable background color).

## Stack

HTML5, vanilla JS, vanilla CSS (custom properties, grid, flex). Inter from Google Fonts. No build step.
