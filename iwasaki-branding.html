Usage Guide — いつ何を使う
IndigoSurfaces, structure, secondary buttons. The brand's "voice."
EmeraldPrimary actions, accents, success states. The brand's "energy."
MistReverse type on dark indigo, hero text, subtle dividers.
LegacyUse original #1A2680 / #1F8C3B only for printed material matching existing collateral.
JP FontIBM Plex Sans JP for body, headings, UI labels. Switch to M PLUS 1 Code when monospace alignment matters (tables, code).
Latin FontIBM Plex Mono for numerics, code, IDs, English navigation.
MixingApply .jp class on elements with Japanese content; Latin tags inherit mono automatically.
