Body & Inline

This is a standard <p> paragraph — the workhorse of body copy. Default text color uses --text-primary on a moss-dark background. Line height is 1.6 for comfortable reading.

Secondary text via .text-secondary — for descriptions, captions, helper copy.

Muted text via .text-muted — for timestamps, hints.

Inline code uses the same font with a subtle background tile. A link looks like this dashed underline and turns lower-opacity on hover.

When to Use What
h1Page title or hero — only one per page.
h2Major sections within a page.
h3Cards, modal titles, or sub-section headers. Uses accent color.
h4Panel eyebrows / category labels (uppercase tracked).
h5 / h6Micro-labels in dense UI — table headers, sidebar groups.
pBody copy. Default. Don't use spans for paragraphs.
spanInline emphasis via .text-accent, .text-secondary, .text-muted.
buttonPrimary for confirm/save, secondary for navigation, ghost for tertiary, icon for toolbar.
aDashed underline — feels handwritten and intentional.
iconsLucide. 14px inside buttons/badges, 20px in nav, 28px for logo.
